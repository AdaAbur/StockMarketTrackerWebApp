# Tradeon - Stock Market Tracker (Web App)

Responsive stock market tracker web app. Part 2 of the Frontend Programming final project.

## How to run

Open `index.html` in a browser. It redirects to the dashboard page.

## Project structure

- `index.html` - entry point (redirects to the dashboard)
- `pages/` - app pages (dashboard, stock details, portfolio)
- `css/` - global styles (theme, base, components, layout)
- `components/` - reusable UI components
- `js/` - scripts
- `assets/` - images and icons

## Technologies

HTML5, CSS3, vanilla JavaScript.
